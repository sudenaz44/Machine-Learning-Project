## 📌 Depodaki Projeler

### 1️⃣ [Car Price ML Pipeline](./Car-Price-ML-Pipeline/)
- Hindistan araç pazarındaki araba verilerini kullanarak satış fiyatlarını tahmin eder  
- Pipeline içerikleri:
  - Veri yükleme ve inceleme
  - Eksik veri temizleme ve dönüştürme
  - Feature engineering (ör. araç yaşı, one-hot encoding)
  - Modeller: Linear Regression, Decision Tree, Random Forest
  - Performans değerlendirmesi (MAE, MSE, R²)
  - Feature importance görselleştirme
- Random Forest modeli ile **R² = 0.97**, **MAE ≈ 70k** ile yüksek doğruluk sağlandı
